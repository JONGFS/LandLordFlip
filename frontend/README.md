# LandlordFlip Frontend

This folder contains the Vite + React frontend for `LandlordFlip`.

## Run locally

```bash
cd frontend
npm install
npm run dev
```

If you add Gemini-powered features, set `GEMINI_API_KEY` in `.env.local`.

## Main files

- `src/App.tsx` for the current product UI
- `src/index.css` for theme and shared styles
- `vite.config.ts` for local dev configuration
